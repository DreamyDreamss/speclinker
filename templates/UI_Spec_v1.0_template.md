# 도메인별 UI 상세 파일 템플릿

> 아래는 `docs/05_설계서/{도메인}/UI_{도메인}.md` 파일에 작성할 형식이다.
> 한 파일에 해당 도메인의 모든 UIS-F 항목을 `## UIS-F-XXX` 앵커로 나열한다.
>
> **도메인 화면 수 기준**: 8개 이하 = 단일 파일 유지 / 9개 이상 = `UI_{도메인}_A.md`, `UI_{도메인}_B.md`로 분할  
> **sl-dev 효율화**: 에이전트가 특정 화면만 필요할 때 `## UIS-F-XXX` 앵커로 grep해 해당 섹션만 읽도록 한다.
